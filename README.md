# ChineseChess
Chinese Chess in Terminal and AI

## Introduction

1. UI in terminal
2. 2 AI agent: Random and Minmax
3. Implemented in VS

## Install and Play

Just open it in Visual Studio (https://visualstudio.microsoft.com/), build and run.
Follow the instructions in terminal.
